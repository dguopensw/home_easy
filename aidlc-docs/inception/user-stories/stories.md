# 유저 스토리 (User Stories)
# 집에 가구 쉽다 (Easy Furniture Fit)

**분류 방식**: 화면(Feature) 기준
**인수 기준 형식**: Given / When / Then
**범위**: MVP + Post-MVP (우선순위 표시)

> 우선순위: 🔴 MVP 필수 / 🟡 MVP UI만 / 🟢 Post-MVP

---

## SCREEN 1 — 홈 화면

### US-01 서비스 소개 확인 🔴
**As a** 처음 방문한 사용자로서
**I want to** 서비스가 무엇인지 한눈에 파악하고 싶다
**So that** 앱을 어떻게 사용하는지 바로 이해할 수 있다

**Acceptance Criteria**:
- Given 사용자가 앱을 처음 실행했을 때
- When 홈 화면이 로드되면
- Then "사진 한 장으로 중고 가구를 내 방에 3D로 배치해보세요" 문구와 3단계 사용법(링크 붙여넣기 → AI 3D 모델 생성 → AR 배치)이 표시된다

### US-02 가구 탐색 시작 🔴
**As a** 중고 가구 구매를 고려하는 민준으로서
**I want to** 홈 화면에서 바로 시작할 수 있고 싶다
**So that** 불필요한 단계 없이 바로 URL을 입력할 수 있다

**Acceptance Criteria**:
- Given 사용자가 홈 화면에 있을 때
- When "지금 시작하기" 버튼을 탭하면
- Then URL 입력 화면으로 이동한다

### US-03 최근 배치 기록 미리보기 🟡
**As a** 앱을 재방문한 사용자로서
**I want to** 이전에 배치해본 가구 기록을 홈에서 바로 볼 수 있고 싶다
**So that** 관심 있는 가구를 빠르게 다시 확인할 수 있다

**Acceptance Criteria**:
- Given 사용자가 홈 화면에 있을 때 (MVP: 더미 데이터 표시)
- When 최근 배치 기록 섹션이 로드되면
- Then 최근 배치한 가구 썸네일 3개가 표시된다
- Then "전체보기" 버튼을 탭하면 배치 기록 화면으로 이동한다

---

## SCREEN 2 — URL 입력 화면

### US-04 중고 가구 URL 입력 🔴
**As a** 당근마켓에서 소파를 발견한 민준으로서
**I want to** 게시글 URL을 붙여넣어 3D 모델을 생성하고 싶다
**So that** 소파가 내 방에 맞는지 AR로 확인할 수 있다

**Acceptance Criteria**:
- Given 사용자가 URL 입력 화면에 있을 때
- When 당근마켓 / 번개장터 / 중고나라 URL을 입력하고 "3D 모델 생성하기"를 탭하면
- Then 로딩 화면으로 이동하고 AI 파이프라인이 시작된다

### US-05 데모 링크로 체험 🔴
**As a** 처음 사용하는 현수로서
**I want to** 내 URL 없이도 앱을 체험해보고 싶다
**So that** 실제 사용 전에 기능을 미리 확인할 수 있다

**Acceptance Criteria**:
- Given 사용자가 URL 입력 화면에 있을 때
- When 데모 링크 중 하나를 탭하면
- Then 해당 URL이 입력 필드에 자동으로 채워진다

### US-06 빈 URL 입력 에러 처리 🔴
**As a** URL을 입력하지 않고 버튼을 누른 사용자로서
**I want to** 명확한 안내를 받고 싶다
**So that** 다음 행동을 바로 알 수 있다

**Acceptance Criteria**:
- Given URL 입력 필드가 비어있을 때
- When "3D 모델 생성하기" 버튼을 탭하면
- Then 입력 필드가 흔들리는 애니메이션이 재생되고 포커스가 이동한다

### US-07 지원하지 않는 플랫폼 URL 에러 🔴
**As a** 지원하지 않는 쇼핑몰 URL을 입력한 사용자로서
**I want to** 지원 여부를 바로 알고 싶다
**So that** 지원되는 플랫폼 URL로 다시 시도할 수 있다

**Acceptance Criteria**:
- Given 당근마켓 / 번개장터 / 중고나라 외의 URL을 입력했을 때
- When "3D 모델 생성하기"를 탭하면
- Then "현재 당근마켓, 번개장터, 중고나라만 지원합니다" 안내 메시지가 표시된다

## SCREEN 3 — 로딩 화면

### US-08 AI 처리 진행 상태 확인 🔴
**As a** 3D 모델 생성을 기다리는 사용자로서
**I want to** 현재 어떤 작업이 진행 중인지 알고 싶다
**So that** 막연히 기다리지 않고 진행 상황을 파악할 수 있다

**Acceptance Criteria**:
- Given 로딩 화면이 시작됐을 때
- When AI 파이프라인이 단계별로 진행되면
- Then 현재 단계명(크롤링 / 이미지 선정 / 치수 측정 / 배경 제거 / 3D 생성)과 진행률(%)이 실시간으로 업데이트된다
- Then 완료된 단계는 체크 표시로 변경된다

### US-09 크롤링 실패 에러 처리 🔴
**As a** 게시글이 삭제된 URL을 입력한 사용자로서
**I want to** 명확한 실패 원인을 알고 싶다
**So that** 다른 URL로 재시도할 수 있다

**Acceptance Criteria**:
- Given 크롤링 단계에서 게시글을 찾을 수 없을 때
- When 크롤링이 실패하면
- Then "게시글을 불러올 수 없습니다. 게시글이 삭제되었거나 URL을 확인해주세요" 메시지와 "다시 시도하기" 버튼이 표시된다

### US-10 3D 생성 실패 에러 처리 🔴
**As a** 3D 생성 중 오류가 발생한 사용자로서
**I want to** 재시도 방법을 알고 싶다
**So that** 다시 처음부터 진행할 수 있다

**Acceptance Criteria**:
- Given TRELLIS 3D 생성 단계에서 오류가 발생했을 때
- When 생성이 실패하면
- Then "3D 모델 생성에 실패했습니다. 다시 시도해주세요" 메시지와 "처음으로" 버튼이 표시된다

## SCREEN 4 — 3D 미리보기 화면

### US-11 3D 모델 인터랙션 🔴
**As a** 소파 3D 모델을 확인하고 싶은 지원으로서
**I want to** 모델을 자유롭게 회전하고 확대해서 보고 싶다
**So that** 가구의 모든 면을 확인할 수 있다

**Acceptance Criteria**:
- Given 3D 미리보기 화면이 로드됐을 때
- When 사용자가 드래그 제스처를 사용하면
- Then 3D 모델이 해당 방향으로 회전한다
- When 사용자가 핀치 제스처를 사용하면
- Then 3D 모델이 확대/축소된다

### US-12 가구 치수 확인 🔴
**As a** 방 치수와 가구 치수를 비교하고 싶은 현수로서
**I want to** 정확한 가로/세로/깊이 수치를 확인하고 싶다
**So that** 방에 들어맞는지 판단할 수 있다

**Acceptance Criteria**:
- Given 3D 미리보기 화면에서 "치수" 탭을 탭했을 때
- When 치수 탭이 활성화되면
- Then 가구의 W(너비) / H(높이) / D(깊이) 수치가 cm 단위로 표시된다
- Then 치수 다이어그램 이미지가 함께 표시된다

### US-13 AR 배치로 이동 🔴
**As a** 3D 모델 확인 후 실제 방에 배치해보고 싶은 민준으로서
**I want to** AR 화면으로 바로 이동하고 싶다
**So that** 실제 공간에서의 크기와 분위기를 확인할 수 있다

**Acceptance Criteria**:
- Given 사용자가 3D 미리보기 화면에 있을 때
- When "AR로 방에 배치하기" 버튼을 탭하면
- Then 카메라 권한 요청 후 AR 배치 화면으로 이동한다

---

## SCREEN 5 — AR 배치 화면

### US-14 AR 가구 배치 및 위치 조작 🔴
**As a** 새 방에 소파를 배치해보고 싶은 현수로서
**I want to** AR 화면에 가구가 바로 나타나고 터치로 위치를 조작하고 싶다
**So that** 실제 공간에서 크기와 위치를 자유롭게 확인할 수 있다

**Acceptance Criteria**:
- Given AR 화면이 시작됐을 때
- When 화면이 로드되면
- Then 별도 탭 없이 3D 가구 모델이 카메라 화면 위에 바로 표시된다
- When 사용자가 가구 모델을 터치하여 드래그하면
- Then 가구가 해당 방향으로 이동한다

### US-15 배치된 가구 조절 🔴
**As a** 가구 위치와 방향을 조정하고 싶은 지원으로서
**I want to** 배치된 가구를 움직이고 회전시키고 싶다
**So that** 가장 잘 어울리는 위치를 찾을 수 있다

**Acceptance Criteria**:
- Given 가구가 AR 공간에 배치됐을 때
- When 두 손가락으로 회전 제스처를 사용하면
- Then 가구가 회전한다
- 참고: 가구 크기 조절 기능은 제공하지 않음 (AI가 추정한 실제 치수 고정)

### US-16 가구 복제 및 삭제 🔴
**As a** 같은 가구를 여러 위치에 놓아보고 싶은 지원으로서
**I want to** 배치된 가구를 복제하거나 삭제하고 싶다
**So that** 다양한 배치 시나리오를 비교할 수 있다

**Acceptance Criteria**:
- Given 가구가 배치됐을 때
- When "복제" 버튼을 탭하면
- Then 동일한 가구가 하나 더 생성된다
- When "삭제" 버튼을 탭하면
- Then 선택된 가구가 제거된다

### US-17 AR 인식 실패 안내 🔴
**As a** 어두운 환경에서 앱을 사용하는 사용자로서
**I want to** AR이 왜 작동하지 않는지 알고 싶다
**So that** 올바른 환경에서 다시 시도할 수 있다

**Acceptance Criteria**:
- Given AR 화면에서 바닥 평면 인식이 되지 않을 때
- When 10초 이상 인식이 안 되면
- Then "조명이 밝은 공간에서 바닥을 향해 천천히 카메라를 움직여주세요" 안내가 표시된다

### US-18 AR 결과 촬영 🔴
**As a** 배치 결과를 저장하고 싶은 사용자로서
**I want to** 현재 AR 화면을 사진으로 찍고 싶다
**So that** 나중에 다시 확인하거나 공유할 수 있다

**Acceptance Criteria**:
- Given 가구가 배치된 AR 화면에서
- When "촬영" 버튼을 탭하면
- Then 현재 화면이 캡처되고 저장 완료 토스트 메시지가 표시된다

## SCREEN 6 — 배치 기록 화면 (Post-MVP)

### US-19 배치 기록 목록 조회 🟢
**As a** 이전에 확인했던 가구를 다시 보고 싶은 민준으로서
**I want to** 지금까지 AR 배치해본 가구 목록을 볼 수 있고 싶다
**So that** 관심 가구를 다시 확인하고 비교할 수 있다

**Acceptance Criteria**:
- Given 로그인된 사용자가 배치 기록 화면에 접근했을 때
- When 화면이 로드되면
- Then 배치한 가구 목록이 썸네일 그리드로 표시된다 (가구명, 방, 날짜 포함)
- When 특정 항목을 탭하면
- Then 해당 가구의 3D 미리보기 화면으로 이동한다

---

## SCREEN 7 — 결과 저장 및 공유 (Post-MVP)

### US-20 AR 배치 결과 공유 🟢
**As a** 배치 결과를 SNS에 올리고 싶은 지원으로서
**I want to** AR 캡처 이미지를 바로 공유하고 싶다
**So that** 팔로워들에게 인테리어 아이디어를 공유할 수 있다

**Acceptance Criteria**:
- Given 결과 화면에 AR 배치 이미지가 표시됐을 때
- When "공유하기" 버튼을 탭하면
- Then 기기의 공유 시트가 열리고 이미지를 SNS / 메시지로 공유할 수 있다

---

## 소셜 로그인 (Post-MVP)

### US-21 카카오 로그인 🟢
**As a** 배치 기록을 저장하고 싶은 사용자로서
**I want to** 카카오 계정으로 간편하게 로그인하고 싶다
**So that** 여러 기기에서 내 배치 기록에 접근할 수 있다

**Acceptance Criteria**:
- Given 비로그인 사용자가 배치 기록 저장을 시도했을 때
- When 카카오 로그인 버튼을 탭하면
- Then 카카오 OAuth 인증 화면이 열린다
- When 인증이 완료되면
- Then 배치 기록이 계정에 저장되고 홈 화면으로 이동한다
